🚀 БЫСТРИЙ СТАРТ (5 ХВИЛИН):

1️⃣  ВСТАНОВІТЬ Node.js (якщо немає)
   → https://nodejs.org/

2️⃣  ВСТАНОВІТЬ ЗАЛЕЖНОСТІ
   → Натисніть на setup.bat в папці проекту
   (або вручну: cd server && npm install && cd ../client && npm install)

3️⃣  ВСТАНОВІТЬ MONGODB
   Вибрайте один із варіантів:
   
   Option A: Локально
   → https://www.mongodb.com/try/download/community
   → Встановіть та запустіть mongod.exe
   
   Option B: В хмарі (рекомендується)
   → https://www.mongodb.com/cloud/atlas
   → Зареєструйтеся та створіть базу
   → Скопіюйте connection string в server/.env

4️⃣  ЗАПУСТІТЬ ДОДАТОК
   
   Terminal 1:
   cd server
   npm start
   
   Terminal 2:
   cd client
   npm start
   
   → Перейдіть на http://localhost:3000

─────────────────────────────────────────
1. Перейти на railway.app
2. Зареєструватись через GitHub

─────────────────────────────────────────
